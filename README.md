# Nyan Video Downloader

A lightweight Python utility to download media files from **YouTube**, **TikTok**, **Douyin**, and **Discord CDN** attachments.

## Features
- Full support for YouTube, TikTok, and Douyin via `yt-dlp`.
- Stream downloads direct file attachments like `cdn.discordapp.net` links.
- Automatically organizes downloads into a local `downloads/` directory.

## Installation
1. Clone this repository:
```bash
git clone [https://github.com/joratheboi2011-beep/nyan-video-downloader.git](https://github.com/joratheboi2011-beep/nyan-video-downloader.git)
cd nyan-video-downloader/video_downloader

Note: If you don't see a downloaded file, type 'pwd' and you will see a GitHub repo installed. Example: C:\Users\YourName\nyan-video-downloader\video_downloader\downloads
