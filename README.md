# Nyan Video Downloader

A lightweight Python utility to download media files from **YouTube**, **TikTok**, **Douyin**, and **Discord CDN** attachments.

## Features
- Full support for YouTube, TikTok, and Douyin via `yt-dlp`.
- Stream downloads direct file attachments like `cdn.discordapp.net` links.
- Automatically organizes downloads into a local `downloads/` directory.
